This tutorial will allow traditional developers that use Android Studio, to start the development of applications in Xamarin Android with Visual Studio.

Знакомство

 
Одним из главных преимуществ мобильной разработки является использование технологий, которые позволяют создавать нативные приложения, использовать большинство технических возможностей операционной системы. Для этого Android Studio предлагает отличный инструмент для построения пользовательского интерфейса Android, такой как Constraint Layout. С другой стороны, у Microsoft есть один из самых популярных и современных языков программирования в настоящее время, C #, и с фантастическим инструментом Xamarin Android, поэтому мы можем создавать нативные приложения, которые используют пользовательский интерфейс Android в Android Studio, кодируя на C # с Помощью Visual Studio.
 
В примере показано построение простого пользовательского интерфейса в Android Studio и кодирование на C# с помощью Visual Studio. Приложение, которое загружает изображение из Интернета, и выводит его на экран Android.
 
Требования
  • Android Studio 3 с Android SDK 6 или более поздней версии.
  • Visual Studio 2017 с Xamarin.

Шаг 1 - Создание приложения в Android Studio

 
Откройте Android Studio и начните новый проект Android Studio,
 
 
 
Поставьте имя и путь к проекту,
 
 
 
Выберите Установленный пакет SDK
 
 
 
Выберите Пустое действие и далее
 
 
 
Нажмите кнопку "Готово"
 
 
 
Вы увидите текст «Hello World» по центру и с экраном с правой стороны с ограничениями пользовательского интерфейса.
 
 
 
Переместите текст вверх и в палитру атрибутов (справа) и измените текст «Hello World» на «Xamarin Android». Дополнительно можно увеличить размер шрифта.
 
 
 
В палитре панели (слева) выберите компонент «ImageView» и поместите его на экран, появившись окна, и выберите «цвет» и «черный», и нажмите «ОК».
 
 
 
На панели атрибутов добавьте идентификатор как "imageView".
 
 
 
Теперь добавьте кнопку, под изображением и поставьте ID "btndownload".
 
 
 
Выберите заголовок, вы увидите точки вокруг с ограничениями по умолчанию.
 
 
 
На изображении выделите точку, которая находится в верхнем центральном углу, и перетащите ее в текст верхней части (заголовок), пока не появится символ подключения. Теперь выделите центральные точки сторон и перетащите их к границам интерфейса, как с правой, так и с левой стороны. Наконец, возьмите центральную точку внизу и перетащите ее в нижнюю часть экрана. Оставаясь следующим:
 
 
 
Проделайте тот же процесс с помощью кнопки.
 
 
 
После завершения сохраните и перейдите в «текстовую» область экрана, чтобы перейти к XML-кодированию интерфейса. Там мы копируем весь код в буфер обмена. Этот код будет скопирован в Visual Studio позже.
 
 
 
Теперь выберите в меню "Инструменты - AVD Manager" и выберите эмулятор, который вы будете использовать. (Вы также можете использовать свое физическое устройство, просто проверьте разрешение, то же самое, которое вы используете в дизайне).
 
 
 
Нажмите на кнопку воспроизведения, появится ваш Android симулятор.
 
 

Шаг 2. Создание приложения в Visual Studio

 
Откройте Visual Studio и запустите новый проект Xamarin для Android
 
 
 
Укажите имя проекта и путь
 
 
 
Выберите Пустое приложение и нажмите «ОК»
 
 
 
Появится следующий экран:
 
 
 
In the name of the project, in the solution explorer, click with the right mouse button and select "Manage Nuget Packages". There you write “Xamarin Android Constraint Layout” and browse, and click on the install button.
 
 
 
Accept the license,
 
 
 
Wait a moment for the packages to be downloaded, usually in about 5 seconds or less.
 
 
 
In the solution explorer, click in the folder "Resources" and "Layout", there is our file "activity_main.axml",
 
 
 
Put the Android Studio XML code in this area (copy and paste), save and compile the solution.
 
 
 
Now, click on the Run, verifying that our emulator appears at the top, being recognized by Visual Studio. Be patient the first time,
 
 
 
With the UI Android works correctly, so now we start the coding.
